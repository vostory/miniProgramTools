一、功能规划（基础版+可扩展）
以“轻量、实用、本地优先”为核心，搭建多模块工具集合，初步规划：

• 基础计算器：满足日常加减乘除、百分比等简单运算；
• 科学计算器：支持三角函数、指数、对数等复杂表达式计算；
• 日期计算器：计算两个日期间隔、日期加减、工作日统计等；
• 单位换算：覆盖长度、重量、温度、面积、体积等常用维度；
• 证件文字识别：身份证、银行卡等信息提取（依托微信官方能力）。
后续将根据反馈，逐步新增更多实用工具模块～

二、开发核心原则
✅ 优先本地运行：尽可能减少网络依赖，核心逻辑本地实现；
✅ 拒绝第三方 API：优先复用微信官方免费能力（如云开发 OCR、本地存储、内置组件）；
✅ 源码全公开：GitHub 按开发阶段提交，方便追溯与协作。

三、技术栈选型（轻量化，降低入门门槛）
前端开发：WXML + WXSS + JavaScript（原生框架）
开发工具：微信开发者工具（官方原生，兼容性最佳）
部署方式：个人小程序账号 + 微信云开发（按需启用）
